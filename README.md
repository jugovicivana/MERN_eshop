# MERN eshop
MERN Shop je cross-platform mobilna e-commerce aplikacija za Android i iOS, razvijena pomoću MERN stack tehnologija i React Native-a. Aplikacija omogućava korisnicima kupovinu proizvoda, upravljanje korpom i narudžbama, dok administratori imaju potpunu kontrolu nad proizvodima, korisnicima i porudžbinama. Projekat je realizovan kao real-world aplikacija, kroz praktično učenje i razvoj kompletne full-stack mobilne aplikacije – od dizajna interfejsa do backend API-ja i baze podataka u oblaku.

## Tehnologije
*Backend*: Node.js, Express.js 
*Frontend*: React Native
*Autentifikacija*: JWT (JSON Web Token)
*Baza podataka*: MongoDB (Cloud baza)

## Ključne funkcionalnosti  
### Proizvodi
- Prikaz svih proizvoda
- Detaljan prikaz proizvoda
- Pretraga proizvoda
- Carousel sa top proizvodima
 
### Korpa i narudžbe  
- Dodavanje i uklanjanje proizvoda iz korpe
- Izmjena količine proizvoda
- Checkout proces (adresa dostave, način plaćanja)
- Kreiranje narudžbe
- Praćenje statusa narudžbi
 
### Korisnički nalozi
- Registracija i prijava korisnika
- JWT autentifikacija
- Pregled ličnih narudžbi
- Korisnički profil
 
### Administracija 
- Dodavanje, izmjena i brisanje proizvoda
- Upravljanje korisnicima
- Pregled svih narudžbi
- Promjena statusa narudžbi
  
## Pokretanje  
### Backend:  
- Otvaranjem aplikacije u nekom code editoru (npr. Visual Studio Code-u), potrebno je pozicionirati se u terminalu u backend direktorijumu i instalirati zavisnosti komandom *npm install*
- Zatim je potrebno podesiti .env fajl sa podacima za konekciju sa MongoDB bazom
- Naredni korak je komanda *npm start* u terminalu (u backend direktorijumu)
 
### Frontend:
- Potrebno je otvoriti frontend direktorijum, pa instalirati zavisnosti pomoću komande *npm install*
- Pokrenuti aplikaciju koristeći komandu *npm start*.
